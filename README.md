# OrbitEye
Projeto desenvolvido para a global Solutioon FIAP 2026.

# Objetivo
O OrbitEye é uma plataforma de monitoramento ambiental inspirada em tecnologias espaciais para auxiliar na prevençao de queimadas, enchentes e impactos climaticos.
